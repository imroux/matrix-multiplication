# About

Multiply 2x Matrices in C++ using Pthreads

> NOTE: tested on Arch Linux with 6.19.8-arch1-1 kernel.\
> Compiler: g++ (GCC) 15.2.1 20260209\
> C++ Standard: C++11

# How to run it

From the project root folder run the following commands:

```
g++ -std=c++11 src/main.cpp src/matrix.cpp -o bin/matrix_multiplication
./bin/matrix_multiplication
```
